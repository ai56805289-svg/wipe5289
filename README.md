# CLEARMAX PRO — 수입차 전용 와이퍼 랜딩페이지

## 프로젝트 개요
BMW, Benz, Audi, VW 등 주요 수입차 67개 차종 전용 와이퍼 판매 랜딩페이지.
차종별 호환 정보 제공 및 문자 주문 연결 기능 포함.

---

## 현재 완성된 기능

### 페이지 구성
| 섹션 | 내용 |
|------|------|
| Hero | 제품 소개, 가격 표시, 문자 주문 버튼 |
| Stats Bar | 호환 차종 수, 브랜드 수, 가격, 배송 |
| Brand Bar | 지원 브랜드 13개 배지 |
| 특장점 | 6가지 제품 특징 카드 (안전 문구 기준) |
| 가격 | 3쌍 20,000원 / 무료배송 / 구성 안내 |
| 차종 호환표 | 67개 차종, 브랜드 필터, 키워드 검색 |
| 주문 방법 | 4단계 주문 안내 |
| FAQ | 5개 아코디언 FAQ |
| CTA | 최종 주문 유도 섹션 |
| Footer | 브랜드 및 주문 안내 (번호 미노출) |

### 문자 주문 기능
- 호환표의 각 차종 "문자로 주문하기" 버튼 클릭 시:
  - 해당 차종명, 어댑터 타입, 사이즈가 문자 본문에 자동 포함
  - sms: 링크로 문자앱 즉시 실행
- 수신 번호는 **JS 코드 내에만 존재** (페이지 화면에 텍스트로 미노출)

### 번호 관리
- `js/data.js` 2번째 줄 `SMS_PHONE_NUMBER` 변수에만 존재
- HTML, CSS 어디에도 번호 텍스트가 노출되지 않음

---

## 파일 구조
```
index.html          ← 메인 페이지 (번호 텍스트 없음)
css/style.css       ← 전체 스타일 (반응형)
js/data.js          ← 67개 차종 데이터 + SMS 설정
js/main.js          ← 검색/필터/FAQ/스크롤 로직
README.md
```

## 가격 정책
- **3쌍 ₩20,000 / 무료배송**
- 뒤 와이퍼 미포함 (앞 와이퍼 전용)
- 추가 수량은 문자 문의

---

## 광고 문구 기준
- 과장·수치 단정 표현 제거 (표시광고법 기준 검토 완료)
- "완벽", "최고", "-30°C" 등 입증 필요 표현 삭제
- 사용 경험 중심 + 안내형 문구로 대체
